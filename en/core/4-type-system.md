# Part 4 — Ontological Type System

For any discussion to be productive, every term used must undergo a "type check." If we attempt to perform logical operations between incompatible types (e.g., treating a C5 fiction as a C1 fact), the system will collapse into a fallacy.

## 4.1 Referent Taxonomy
We classify each term according to the nature of what it points to within the layers (C0-C5):

| Type | Label | Operational Description | Example |
| :--- | :--- | :--- | :--- |
| **RAW** | `v0` | The object in C0, before being measured or named. The "thing-in-itself." | The spectrum before Newton. |
| **ACCESSIBLE** | `v1` | C1/C2 phenomena verified by the Pipeline. These have predictive power. | Gravity, oxygen, voltage. |
| **CONVENTION** | `v2` | C4/C5 constructions with authors or social consensus. They are "correct" but not "true." | Money, language, GDP. |
| **INDEFINITE** | `v?` | Concepts undergoing validation or lacking sufficient C1 data. | Wormholes, consciousness. |
| **DANGLING** | `v-err` | Dangling pointers. Words that lack a referent in any layer or are purely circular. | Destiny, luck, "the system" (as an entity). |

## 4.2 Instance vs. Class Rule
A common error is confusing a **class** (a C4 label) with an **instance** (a C1 object).
* **Instance (C1):** The specific dog that bit you (a physical object with mass and behavior).
* **Class (C4):** "Canine dangerousness" (a social abstraction).
Operating on the class as if it were an instance is a type failure that results in inefficient public policies and circular debates.

## 4.3 The Labeling Algorithm
Before processing an argument, every noun must be subjected to the following test:

1.  **Does it have measurable mass, energy, or information?** -> Mark as `v1`.
2.  **Would it disappear if all humans stopped believing in it?** -> Mark as `v2`.
3.  **Does the definition rely exclusively on other words without touching C1?** -> Mark as `v-err`.

## 4.4 Type Mismatch
Most social conflicts occur due to an undetected type mismatch.
* **Type Error A:** Treating a `v2` (Social Fiction) as a `v1` (Physical Reality). *Example: "The economy must grow by natural law."*
* **Type Error B:** Attempting to modify a `v1` (Physical Reality) using `v2` tools (Rhetoric). *Example: "We decree that gravity is oppressive."*

## 4.5 The "Dangling" Status
A `v-err` term is radioactive to knowledge. If a conversation includes more than 20% dangling pointers, the framework recommends aborting the communication. In such cases, information is not being exchanged; instead, participants are merely synchronizing emotional states (C3) without an anchor.

---
*This type system makes the framework auditable: if a term changes its type, there must be a commit justifying the new evidence in C1.*
