# Framework Epistémico: Realidad, Lenguaje y Conocimiento (v1.0)

> **Herramientas para clasificar términos, evaluar argumentos y distinguir conocimiento de opinión.**

## 🎯 ¿Qué es esto?
Este es un framework epistemológico que opera en el nivel del **metalenguaje**. Su función no es proponerte una "nueva verdad", sino proporcionarte una infraestructura diagnóstica para evaluar las estructuras que otros sistemas usan para hacer afirmaciones.

Su método es la **ingeniería inversa**: parte del hecho verificable de que la tecnología funciona para construir definiciones operacionales de Realidad y Conocimiento que sean consistentes con ese éxito material.

## 💎 El Axioma Fundacional
Este framework se sostiene sobre una evidencia mínima e innegable: **Usted está leyendo este texto en un dispositivo electrónico que funciona.**

Para que esto sea posible, miles de principios físicos deben ser verdaderos simultáneamente con una precisión extrema. Si esos fundamentos fueran falsos, este dispositivo nunca habría podido existir. Partimos de este éxito tecnológico como ancla de realidad para desmontar el relativismo lingüístico.

## 🏗️ Estructura del Sistema
El núcleo del proyecto se divide en cinco pilares:

* **Ontología por Capas (C0-C5):** Un mapa que distingue desde la Realidad Raw (C0) hasta las Instituciones Formales (C5).
* **Epistemología Operacional:** La unificación de los conceptos de "Verdad" y "Conocimiento" bajo el criterio de verificación fáctica.
* **Pipeline de Validación:** Un flujo de 8 filtros secuenciales por los que debe pasar una afirmación para ser considerada conocimiento en sentido fuerte.
* **Sistema de Tipos Ontológicos:** Herramienta para etiquetar referentes como RAW, ACCESSIBLE, FICCIÓN, INDEFINIDO o DANGLING.
* **Protocolo de Pre-debate:** Un método de 4 pasos para eliminar la ambigüedad semántica antes de iniciar cualquier discusión.

## 🚀 Estatus del Proyecto
Este framework se publica como una **hipótesis abierta**. Bajo sus propios criterios, no reclama un estatus de verdad final que no haya ganado. Su validez se mide por su utilidad para diagnosticar confusiones del lenguaje y predecir fallos en sistemas sociales.

## 🤝 Cómo Contribuir
Este es un proyecto **Open Source**. Se buscan colaboradores para:
* **Reporte de "Bugs" Lógicos:** Si detectas una inconsistencia en la jerarquía o un error de categoría.
* **Casos de Estudio:** Aplicar el framework a dominios específicos (economía, derecho, biología).
* **Traducciones:** Actualmente en español, inglés y portugués.

Al colaborar, por favor utiliza nuestras **Issue Templates** para identificar en qué capa (C0-C5) detectaste el problema.
