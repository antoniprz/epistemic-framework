# Framework Epistémico: Realidad, Lenguaje y Conocimiento (v1.0)

> **Herramientas para clasificar términos, evaluar argumentos y distinguir conocimiento de opinión.**

## 🎯 ¿Qué es esto?
Este es un framework epistemológico que opera en el nivel del **metalenguaje**. Su función no es proponerte una "nueva verdad", sino proporcionarte una infraestructura diagnóstica para evaluar las estructuras que otros sistemas usan para hacer afirmaciones.

[cite_start]Su método es la **ingeniería inversa**: parte del hecho verificable de que la tecnología funciona para construir definiciones operacionales de Realidad y Conocimiento que sean consistentes con ese éxito material.

## 💎 El Axioma Fundacional
Este framework se sostiene sobre una evidencia mínima e innegable: **Usted está leyendo este texto en un dispositivo electrónico que funciona.**

Para que esto sea posible, miles de principios físicos deben ser verdaderos simultáneamente con una precisión extrema. Si esos fundamentos fueran falsos, este dispositivo nunca habría podido existir. [cite_start]Partimos de este éxito tecnológico como ancla de realidad para desmontar el relativismo lingüístico[cite: 47, 48].

## 🏗️ Estructura del Sistema
El núcleo del proyecto se divide en cinco pilares:

1.  [cite_start]**Ontología por Capas (C0-C5):** Un mapa que distingue desde la Realidad Raw (C0) hasta las Instituciones Formales (C5)[cite: 56, 26, 27].
2.  [cite_start]**Epistemología Operacional:** La unificación de los conceptos de "Verdad" y "Conocimiento" bajo el criterio de verificación fáctica[cite: 56, 52].
3.  [cite_start]**Pipeline de Validación:** Un flujo de 8 filtros secuenciales por los que debe pasar una afirmación para ser considerada conocimiento en sentido fuerte[cite: 53, 56].
4.  [cite_start]**Sistema de Tipos Ontológicos:** Herramienta para etiquetar referentes como RAW, ACCESIBLE, FICCIÓN, INDEFINIDO o DANGLING[cite: 56, 102].
5.  [cite_start]**Protocolo de Pre-debate:** Un método de 4 pasos para eliminar la ambigüedad semántica antes de iniciar cualquier discusión[cite: 105, 59].

## 🚀 Estatus del Proyecto
Este framework se publica como una **hipótesis abierta**. Bajo sus propios criterios, no reclama un estatus de verdad final que no haya ganado. [cite_start]Su validez se mide por su utilidad para diagnosticar confusiones del lenguaje y predecir fallos en sistemas sociales[cite: 113].
